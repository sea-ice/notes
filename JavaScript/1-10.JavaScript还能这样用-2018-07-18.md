#### 扁平化数组

1. 使用toString。举例：

```javascript
[1,[2,3,4],4,[[4,5]]].toString().split(',')
// ["1", "2", "3", "4", "4", "4", "5"]
```

2. 使用数组的reduce方法

```javascript
let flatten = arr => arr.reduce((a, b) => a.concat(Array.isArray(b) ? flatten(b) : b), [])
let result = flatten([1,[2,3,4],4,[[4,5]]])
// [].concat([1, 2]) // [1, 2]
// [].concat(3) // [3]
```
